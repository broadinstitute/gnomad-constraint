# Matched pLoF o/e by Missense Score Percentile

## Question

Do missense prediction scores capture constraint signal beyond gene-level pLoF constraint? Genes with low pLoF o/e tend to also be depleted for missense variation, so when a missense score ranks variants in constrained genes as more pathogenic, is that just reflecting gene-level LoF intolerance rather than missense-specific signal?

## Approach

For each missense score percentile bin (1--100, where 100 is most pathogenic), we compute two quantities:

1. **Missense o/e** (solid lines): The observed/expected ratio for missense variants in that bin, using `obs / exp_with_adj_r` from the gnomAD v4.1 constraint pipeline.

2. **Matched pLoF o/e** (dashed lines): A weighted average of gene-level pLoF o/e across all genes contributing variants to that bin. This represents the pLoF constraint "background" for the gene composition of each percentile bin.

   Formally: `matched_plof_oe = sum(plof_obs_g * w_g) / sum(plof_exp_g * w_g)` where the sum is over genes *g* contributing to the bin.

Two weighting approaches are compared (neither uses observed missense counts, which would be circular):

- **Weight by adjusted expected missense** (`expected_variants * adj_r`): Consistent with the missense o/e denominator.
- **Weight by possible missense variants**: Uses the count of possible missense sites, independent of mutation rate modeling.

If a missense score captures signal beyond gene-level pLoF, the missense o/e (solid) should drop faster than the matched pLoF o/e (dashed). The gap between them is the missense-specific constraint signal.

## Plot 1: Missense o/e only (baseline)

Replicates the original plot showing missense depletion by score percentile for three scores.

![Missense o/e by score percentile](missense_oe_unmatched_plot.png)

- All three scores show increasing missense depletion at higher (more pathogenic) percentiles.
- AlphaMissense and PopEVE show steeper drops at the highest percentiles than ESM1v.
- Dashed reference lines show global synonymous o/e (~1.05) and pLoF o/e (~0.58).

## Plot 2: Matched pLoF o/e weighted by adjusted expected missense count

![Matched pLoF weighted by expected](matched_plof_oe_by_expected.png)

- **Solid lines**: missense o/e (same as Plot 1).
- **Dashed lines**: matched pLoF o/e, weighted by `expected_variants * adj_r`.
- The matched pLoF o/e also decreases at higher percentiles, confirming that high-scoring bins are enriched for LoF-intolerant genes.
- However, the missense o/e drops much more steeply, demonstrating that all three scores capture missense-specific constraint signal beyond gene-level pLoF.

## Plot 3: Matched pLoF o/e weighted by possible missense variants

![Matched pLoF weighted by possible](matched_plof_oe_by_possible.png)

- Same layout as Plot 2 but with possible missense variants as the weight.
- Results are very similar to the expected-weighted version.

## Plot 4: Side-by-side comparison of both weighting approaches

![Comparison of weighting approaches](matched_plof_oe_comparison.png)

- Left: weighted by adjusted expected missense count. Right: weighted by possible missense variants.
- The two approaches yield nearly identical results, confirming the finding is robust to the choice of weight.

## Script

The plotting script is `gnomad_constraint/plots/plot_matched_plof.py`. The matched pLoF data was generated by `gnomad_constraint/plots/determine_missense_score_percentiles.py` using the `--export-matched-plof-summary` flag.
